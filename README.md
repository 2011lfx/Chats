# Chats

An umbrella project

## Components

### Clients

* [Chats-iPhone][1]

### Servers

* [Chats-Server-REST][2]


## Demo Flow

1. Sign up
    * Entor phone number
    * Verify phone number
    * Fill out profile
        * picture
        * first name
        * last name
2. Log out
3. Log in
    * Enter phone number
    * Verify phone number
4. Delete account

Accounts may be deleted arbitrarily.
