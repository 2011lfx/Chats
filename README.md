#### Explore

1. Sign up
    * Enter phone number
    * Verify phone number
    * Fill out profile
        * picture (incomplete)
        * first name
        * last name
2. Log out
3. Log in
    * Enter phone number
    * Verify phone number
4. Delete account

Accounts may be deleted arbitrarily.
